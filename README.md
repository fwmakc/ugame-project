# Шаблон для игр на движке UGame

Шаблон поддерживает мультиплатформенную разработку приложений под Web / Windows / MacOS / Linux / Android / IOS.

- TypeScript, Vite,
- сборка под desktop с помощью Electron или Tauri,
- сборка под mobile с помощью Capacitor,
- автопроверки и форматирование ESLint, Prettier,
- автотесты Vitest,
- модули ECMAScript (вместо CommonJS),
- стандарт ES2022.

# Начало работы

Выполняем

```
yarn create ugame
# npm create ugame
```

После объявления можно указать название проекта, например:

```
yarn create ugame my game
```

Название проекта будет автоматически преобразовано:

```
name: "my-game",
productName: "My Game"
```

Проект будет установлен в папку с названием проекта.

В данном примере:

```
./my-game
```
